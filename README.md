# Spring Boot Testing - Demo / Proof of Concept

This repository contains code for a proof of concept and serves as a reference for implementing testing in a Spring Boot application.

## Overview

The repository is part of the **IET Result Generation Software System Project** and focuses on demonstrating various testing approaches in Spring Boot.
